# Exercício 1 - Parte 5: Projeto Final - Mini Portfólio Pessoal

## 1. Objetivo

O objetivo deste projeto é consolidar os conhecimentos adquiridos nas partes anteriores do Exercício 1, aplicando conceitos de HTML semântico, links, imagens e uma pequena interação JavaScript para criar um mini portfólio pessoal. O projeto visa demonstrar a capacidade de estruturar uma página web de forma organizada e acessível.

## 2. Estrutura de Pastas

A estrutura de pastas do projeto é organizada da seguinte forma:

```
frontend/
└── Exercicio 1/
    └── Parte 5 - Projeto Final/
        ├── index.html
        ├── css/
        │   └── style.css
        ├── js/
        │   └── script.js
        ├── img/
        │   └── perfil.jpg
        └── README.md
```

*   **`index.html`**: O arquivo principal que contém a estrutura e o conteúdo da página web, utilizando tags HTML semânticas como `<header>`, `<nav>`, `<main>`, `<section>` e `<footer>`.
*   **`css/style.css`**: Arquivo de folha de estilos (CSS) para estilizar a página. Atualmente, está vazio, mas pronto para receber estilos personalizados.
*   **`js/script.js`**: Arquivo de script JavaScript para adicionar interatividade à página. Contém uma função simples para exibir uma saudação ao clicar em um botão.
*   **`img/`**: Pasta destinada a armazenar imagens utilizadas no projeto, como a imagem de perfil (`perfil.jpg`).
*   **`README.md`**: Este arquivo, que fornece informações detalhadas sobre o projeto.

## 3. Como Abrir o Projeto

Para visualizar o projeto em seu navegador, siga os passos:

1.  **Baixe e descompacte** o arquivo `Parte_5_Projeto_Final.zip`.
2.  Navegue até a pasta `frontend/Exercicio 1/Parte 5 - Projeto Final`.
3.  Abra o arquivo `index.html` diretamente em seu navegador de preferência (clicando duas vezes nele ou arrastando-o para a janela do navegador).

## 4. Validação e Revisão

O `index.html` foi construído com as seguintes considerações:

*   **HTML Semântico**: Utilização de tags como `<header>`, `<nav>`, `<main>`, `<section>` e `<footer>` para garantir uma estrutura clara e significativa para os navegadores e tecnologias assistivas.
*   **Acessibilidade**: A imagem possui o atributo `alt` descritivo (`alt="Foto de perfil genérica"`), e os links são claros em seu propósito.
*   **Links**: Inclui um link externo para o repositório no GitHub e links internos para as seções da própria página.
*   **JavaScript**: Uma pequena interação foi adicionada através de um botão que exibe um alerta de saudação, demonstrando a integração básica de JavaScript.

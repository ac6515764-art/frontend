# Exercicio 1 - Parte 1: Pesquisa sobre Fundamentos de Frontend

## O que é Frontend?

**Frontend** refere-se à parte de um website ou aplicação com a qual o usuário interage diretamente. É tudo o que o usuário vê e experimenta no navegador, incluindo o layout, as cores, as fontes, as imagens e a interatividade. O desenvolvimento frontend foca na experiência do usuário (UX) e na interface do usuário (UI), garantindo que a aplicação seja visualmente atraente, responsiva e fácil de usar [1].

## O que é Cliente x Servidor?

O modelo **Cliente-Servidor** é uma arquitetura de rede onde o cliente (geralmente o navegador web do usuário) solicita recursos ou serviços de um servidor (um computador remoto que armazena e entrega esses recursos). O cliente envia requisições (por exemplo, para carregar uma página web) e o servidor responde a essas requisições, enviando os dados necessários de volta ao cliente. O frontend é executado no lado do cliente, enquanto o backend (ou servidor) lida com a lógica de negócios, banco de dados e autenticação [2].

## Qual é o papel de HTML/CSS/JavaScript?

*   **HTML (HyperText Markup Language)**: É a linguagem de marcação padrão para criar páginas web. Ele define a estrutura e o conteúdo de uma página, utilizando elementos como títulos, parágrafos, imagens e links.
*   **CSS (Cascading Style Sheets)**: É a linguagem utilizada para estilizar a aparência de um documento HTML. Ele controla o layout, as cores, as fontes e outros aspectos visuais, separando a apresentação do conteúdo.
*   **JavaScript**: É uma linguagem de programação que permite adicionar interatividade e dinamismo às páginas web. Com JavaScript, é possível criar animações, validar formulários, manipular o DOM (Document Object Model) e interagir com APIs.

Juntos, HTML, CSS e JavaScript formam a base de qualquer interface frontend, trabalhando em conjunto para construir e apresentar o conteúdo de forma interativa e visualmente agradável.

## O que é W3C?

A **W3C (World Wide Web Consortium)** é a principal organização internacional de padronização para a World Wide Web. Fundada por Tim Berners-Lee, o inventor da Web, a W3C desenvolve padrões abertos para garantir a interoperabilidade e o crescimento a longo prazo da Web. Isso inclui a padronização de tecnologias como HTML, CSS, XML e muitas outras, promovendo a acessibilidade e a qualidade da web para todos os usuários [3].

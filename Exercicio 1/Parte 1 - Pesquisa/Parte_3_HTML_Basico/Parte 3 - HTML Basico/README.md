# Exercício 1 - Parte 3: HTML Básico

Este `README.md` explica as tags HTML utilizadas no arquivo `index.html` desta seção do projeto.

## Tags HTML Utilizadas

*   **`<h1>` (Heading 1)**: Utilizada para definir o título mais importante de uma página ou seção. Deve haver apenas um `<h1>` por página para garantir a semântica e a acessibilidade.

*   **`<h2>` (Heading 2)**: Utilizada para definir subtítulos de segunda importância, hierarquicamente abaixo do `<h1>`. Ajuda a organizar o conteúdo em seções lógicas.

*   **`<p>` (Paragraph)**: Representa um parágrafo de texto. É a tag fundamental para blocos de texto em uma página HTML.

*   **`<ul>` (Unordered List)**: Cria uma lista não ordenada de itens. Os itens da lista são geralmente exibidos com marcadores (bolinhas, quadrados, etc.).

*   **`<ol>` (Ordered List)**: Cria uma lista ordenada de itens. Os itens da lista são geralmente exibidos com numeração ou letras, indicando uma sequência.

*   **`<li>` (List Item)**: Representa um item dentro de uma lista (`<ul>` ou `<ol>`). Cada `<li>` deve estar contido dentro de uma dessas tags de lista.
